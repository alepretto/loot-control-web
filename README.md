# Loot Control Web

Frontend do **Loot Control** — sistema de gestão financeira pessoal.

## Stack

- **Next.js 15** (App Router) + TypeScript
- **TailwindCSS** — styling
- **Bun** — package manager

## Setup

```bash
bun install
bun dev
```
